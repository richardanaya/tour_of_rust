Module

Jedes Programm bzw. jede Bibliothek in Rust ist ein crate (Kiste).

Jedes crate besteht aus einer hierarchischen Ansammlung von modules.

Programme werden aus einem root module (Wurzelelement) eines crates gestartet, die Funktion main() in einer Datei namens "main.rs" bildet den Startpunkt.

Bibliotheken haben ein root module in der Datei "lib.rs" und exportieren nützliche Elemente.

Um komplette Pfade wie std::f64::consts::PI austippen zu müssen, wurde das use Schlüsselwort eingeführt, um die Verwendung von Attributen zu erleichtern. Beispielsweise erlaubt use std::f64::consts::PI uns den Identifier PI in der main-Funktion zu verwenden.

std ist das crate der Standardbibliothek von Rust, das allerlei nützliche Datenstrukturen und Funktionen beinhalten, durch welche wir mit dem Betriebssystem kommunizieren können.

Eine nützliche Ressource, um crates zu finden, bildet https://crates.io, die von einer Fanbase aufrechterhalten wird.
