Sempre usar Option/Result posse devenir Ã­ncomod quande on simplicmen vole scrir un poc code. Ambi Option e Result have un metode nominat unwrap (aperter) quel es comod por strax obtenir un valore ma con un poc desbellesse. unwrap va:

  1. trovar li valore intra Option/Result
  2. si li valore es de tip None/Err, panic!

Ti du pezzes de code es egal:

mi_option.unwrap()

match mi_option {
  Some(v) => v,
  None => panic!("alquel erra-missage generat de Rust!")
}

Similmen:

mi_result.unwrap()

match mi_result {
  Ok(v) => v,
  Err(e) => panic!("alquel erra-missage generat de Rust!")
}

Esse un bon Rustaceane e usa match sempre quande tu posse!
