Basistypen

Rust besitzt eine vielzahl an bekannten Datentypen

  • booleans - bool für true/false bzw. wahr/falsch Werte
  • unsigned integers - u8u32u64 für positive Ganzzahlen
  • signed integers - i8i32i64 für vorzeichenbehaftete Ganzzahlen
  • pointer sized integers - uintisize für Indizes und Größen im Speicher
  • floating point - f32f64
  • text zeug - strchar
  • tuple - (value,value,...) zum Übergeben von Wertsequenzen auf dem Stack
  • slices - &[T] um Views in Sequenzen zu referenzieren (hier evt bessere Übersetzung finden)

Wir werden im späteren Verlauf vereinzelt genauer darauf eingehen.

Eine Warnung vorweg: text-Zeug wird etwas komplexer sein als du es vielleicht von anderen Programmiersprachen gewohnt bist, da Rust eine Systemprogrammiersprache ist und daher mehr auf effektives Speichermanagement fokussiert. Kein Panik auf der Titanic aber, wir werden unser Bestes geben, euch darauf vorzubereiten!

Noch ein Hinweis: Numerische Typen können auch nach dem Zahlenliteral spezifiziert werden (z.B. 13u32)
